# Horses

A horse-themed browser game.

## Prerequisites

- Node.js (via nvm): `nvm use --lts`

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Opens local server at http://localhost:5173

## Testing

```bash
npm test
```

## Build

```bash
npm run build
```

Output in `dist/` directory.

## Preview production build

```bash
npm run preview
```
